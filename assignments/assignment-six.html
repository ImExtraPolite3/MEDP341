Week Six Response

Using a Joke API I created a Random Joke Generator. I first made an async function that will show an initial joke when the page is opened. I created another async function that changes the joke displayed depending on the button you click. You have four different button with different topics and when you click on any the second async retrieves a new joke pertaining to that topic.
