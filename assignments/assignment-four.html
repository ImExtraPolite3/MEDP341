Week Four Response

The website that I decided to go with was Click To Confirm.

This website makes great use of the <details> and <summary> tags in order to hide and display the text.

When looking at the page you quickly realize that you are looking at an illusion. The lines look curved however they are all straight. This illusion was made using css-gradient on the summary tags. With the use the css function "calc()" this allows the creator to precisly put the squares in areas that would perpetuate the illusion.

The JavaScript is what is used to dynamically create the tags as well as the hidden text that is also dynamically created using a set of words from predefined lists. It also allows the user to click on any of the sections in order to see the hidden text. Each time a new section is clicked, other sections close automatically.
